# Source Members

## DATECONV.SQLRPGLE

SQL RPGLE source for the DATECONV service program.

### Contains

- Six exported date conversion procedures
- One internal logging procedure (`Log_Conversion`)
- Embedded SQL used to write conversion results to the `DATE_CONVERSIONS` table

### Compiled Using

```cl
CRTSQLRPGI OBJ(YOURLIB/DATECONV)
   SRCFILE(YOURLIB/QRPGSQLSRC)
   SRCMBR(DATECONV)
   OBJTYPE(*MODULE)
   COMMIT(*NONE)
   CLOSQLCSR(*ENDMOD)
```

---

## DATECONV.BND

Binder source used during service program creation.

Defines which procedures are exported and available to external programs.

### Referenced By

```cl
CRTSRVPGM SRVPGM(YOURLIB/DATECONV)
   MODULE(YOURLIB/DATECONV)
   SRCFILE(YOURLIB/QSRVSRC)
   SRCMBR(DATECONV)
```

---

## TESTDCONV.RPGLE

Consumer program used to test the DATECONV service program.

Initializes today's date in YMD, MDY, and DMY formats and invokes each exported procedure.

### Compiled Using

```cl
CRTBNDRPG PGM(YOURLIB/TESTDCONV)
   SRCFILE(YOURLIB/QRPGLESRC)
   SRCMBR(TESTDCONV)
```

## DATE_CONVERSIONS (*FILE)

SQL table used to log conversion results generated by the DATECONV service program.

### Created Using

```sql
CREATE TABLE YOURLIB.DATE_CONVERSIONS
(
  Conversion_Type CHAR(10) NOT NULL,
  Date_In         CHAR(10) NOT NULL,
  Date_Out        CHAR(10) NOT NULL,
  Created_Ts      TIMESTAMP NOT NULL
                  DEFAULT CURRENT TIMESTAMP
);
```
